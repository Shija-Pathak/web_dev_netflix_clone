# Netflix Clone - Vanilla JavaScript

A pixel-perfect Netflix homepage clone built with pure HTML, CSS, and JavaScript.

## Features

✅ Auto-playing hero video with muted loop
✅ Play button opens video player modal with controls
✅ More Info button opens movie details modal
✅ Two category rows with horizontal sliders
✅ Left/right arrow navigation for movie rows
✅ Hover enlarge effect on movie cards
✅ Fully responsive design (mobile, tablet, desktop)
✅ Complete header with logo, navigation, search, and profile
✅ Comprehensive footer with social links

## Folder Structure

```
netflix-clone/
├── index.html          # Main HTML file
├── styles.css          # All styling
├── app.js              # JavaScript functionality
├── README.md           # Documentation
└── assets/
    ├── images/         # Movie poster images (12+ placeholders included)
    └── video/          # Hero video (hero.mp4)
```

## How to Run

1. Open `index.html` directly in your browser
2. No server or build tools required
3. All assets use relative paths

## Video Assets

Place your `hero.mp4` video file in `assets/video/hero.mp4`
- The same video is used for both hero background and video player modal
- Recommended: 1920x1080 resolution, MP4 format

## Image Assets

Movie posters are generated as SVG placeholders with unique colors and titles
- 14 unique movie cards across 2 categories
- Each card contains title, description, genre, and year metadata
- Replace SVG data URIs with actual images by updating the `src` attributes

## Responsive Breakpoints

- Desktop: 1024px and above
- Tablet: 768px - 1023px
- Mobile: 480px - 767px
- Small Mobile: below 480px

## Browser Compatibility

Works on all modern browsers:
- Chrome, Firefox, Safari, Edge
- Mobile browsers (iOS Safari, Chrome Mobile)
